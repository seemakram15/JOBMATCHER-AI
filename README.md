# Jobmatcher

Jobmatcher helps job seekers discover better-fit roles, manage their CV profile, and track applications from one workspace.

## Services

- Account signup, signin, and password recovery
- Profile and job-preference management
- CV upload, parsing, and profile extraction
- Skill management with editable proficiency ranking
- Live job discovery and role matching
- Job filtering by role, skills, location, salary, work mode, job type, and experience
- Saved jobs and application tracking
- Kanban application tracker with status updates
- CSV export for application history
- Admin source-health monitoring
